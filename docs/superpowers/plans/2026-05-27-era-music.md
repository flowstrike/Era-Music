# Era Music Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an offline Material 3 music player ("Era Music") with library browsing, playlists, favorites, search/sort, background playback, and a now-playing/mini-player UI.

**Architecture:** Single-Activity Jetpack Compose app, MVVM. Audio comes from `MediaStore`. Playback runs in a Media3 `MediaSessionService` (ExoPlayer); the UI binds a `MediaController` via a `PlayerConnection` that exposes state as `StateFlow`. Playlists/favorites persist in Room; sort prefs in DataStore. Dependencies wired manually through an `AppContainer` on the `Application` (no Hilt).

**Tech Stack:** Kotlin, Compose + Material 3, Media3 (ExoPlayer + session), Room (KSP), DataStore, Navigation-Compose, Coil, Coroutines.

## Task 4: Playback service + PlayerConnection

**Files:** Create `playback/MusicService.kt`, `playback/PlayerConnection.kt`, `playback/SleepTimer.kt`; modify `AndroidManifest.xml`.

- [ ] `MusicService : MediaSessionService` ‚Äî builds `ExoPlayer`, wraps in `MediaSession`; `onGetSession` returns it; release in `onDestroy`. Set `setAudioAttributes(music, handleAudioFocus=true)` and `setHandleAudioBecomingNoisy(true)`.
- [ ] Manifest: add service with `android:foregroundServiceType="mediaPlayback"`, `android:exported="true"` + `<intent-filter><action android:name="androidx.media3.session.MediaSessionService"/></intent-filter>`. Add permissions: INTERNET, READ_MEDIA_AUDIO, READ_EXTERNAL_STORAGE (maxSdk=32), POST_NOTIFICATIONS, FOREGROUND_SERVICE, FOREGROUND_SERVICE_MEDIA_PLAYBACK. Set `android:name=".EraApp"` on `<application>`.
- [ ] `PlayerConnection(context)`: builds `MediaController` async (addListener, directExecutor); exposes `StateFlow`s: `currentMediaId`, `isPlaying`, `positionMs`, `durationMs`, `shuffle`, `repeatMode`. Position updated by a 500ms coroutine loop while playing. Methods: `setQueue(songs, startIndex)`, `playPause()`, `next()`, `prev()`, `seekTo(ms)`, `toggleShuffle()`, `cycleRepeat()`. Maps `Song`‚Üí`MediaItem` (uri + metadata title/artist/artworkUri from album art).
- [ ] `SleepTimer`: `start(durationMs, onFire)`, `cancel()`, `remainingMs: StateFlow`. Pure coroutine countdown ‚Äî unit-testable with a test dispatcher.
- [ ] **Test:** `SleepTimerTest` using `runTest` + virtual time: start(1000), advance, assert onFire called once and remaining hits 0. `./gradlew :app:testDebugUnitTest --tests "*SleepTimerTest"`.
